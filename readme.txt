HOW TO
1.	Install php7.0 (if you use a high version of php Laravel will require the Mcrypt PHP extension)
2.	Install composer, then run the command below after composer successfully gets installed
	-	Composer update self, this installs all the dependencies
Install NPM Dependencies, run the command below
	-	npm install
3.	Clone the application from git, using the link below
	-	git clone -b https://github.com/TMwansambo/Taxpayer_MRA.git
4.	Navigate to the application folder
5.	Then run the command below to execute the application
	-	Php artisan serve
Make sure you have internet connection in order to consume the API.
